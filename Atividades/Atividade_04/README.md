## Atividade 04

- Desenvolva um programa que crie um objeto da classe veiculo
    (pode ser qualquer veiculo) com os atributos:

  - Fabricante
  - Modelo
  - Placa
  - Ano 
  - Cor

O Programa deverá mostrar os dados do veículo (obs: veículo não motra dados, por tatno não pode ser
via método)

- Atenção: O usuário deverá informas os dados do veículo.
