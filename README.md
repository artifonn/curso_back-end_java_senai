# ☕ Curso de Desenvolvimento Back-End Java

Bem-vindo ao repositório oficial das minhas atividades e projetos do **Curso de Desenvolvimento Back-End com Java**. 

Este repositório documenta a minha jornada de aprendizado, desde os fundamentos da programação até a construção de aplicações web completas e integradas com banco de dados. Aqui você encontrará exercícios práticos, desafios e os projetos finais de cada módulo.

---

## 🎯 Objetivo do Curso
Desenvolver competências completas para a criação de sistemas back-end robustos utilizando a linguagem Java e o ecossistema Spring, integrando conceitos de orientação a objetos, persistência de dados e construção de interfaces web responsivas.

---

## 📚 Ementa e Trilhas de Aprendizado

Este repositório está organizado de acordo com a progressão das disciplinas do curso:

### 1. Fundamentos e Lógica
* **Lógica de Programação com Java:** Variáveis, tipos de dados, estruturas de controle (if/else, loops) e algoritmos básicos.
* **Versionamento de Código:** Controle de histórico, branchs e colaboração utilizando **Git** e **GitHub**.
* **Programação Orientada a Objetos (POO):** Classes, objetos, encapsulamento, herança, polimorfismo e interfaces.

### 2. Desenvolvimento Front-End Básicos
* **Estrutura e Estilização:** Criação de interfaces web estruturadas com **HTML5** e estilizadas com **CSS3**.
* **Dinamismo e Layout:** Interatividade com **JavaScript** e criação de layouts responsivos utilizando o framework **Bootstrap**.

### 3. Banco de Dados e Persistência
* **Modelagem de Dados:** Criação de Diagramas de Entidade-Relacionamento (DER).
* **Banco de Dados Relacional:** Linguagem SQL e gerenciamento de dados utilizando o **MySQL**.
* **Mapeamento Objeto-Relacional (ORM):** Integração do Java com o banco de dados utilizando **JPA** e **Hibernate** para persistência de dados eficiente.

### 4. Frameworks e Desenvolvimento Web (Avançado)
* **Interfaces Gráficas (GUI):** Criação de aplicações desktop com Java (ex: Swing/JavaFX).
* **Spring Framework:** Construção da arquitetura da aplicação, injeção de dependências e configuração do back-end.
* **Spring Boot:** Desenvolvimento ágil e simplificado de aplicações web.
* **Thymeleaf:** Motor de templates para renderização dinâmica de páginas HTML integradas ao Spring.
